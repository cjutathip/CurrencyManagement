# Project Overview

This project is a Currency Management System developed with ASP.NET Core MVC (.NET 8).

The application allows users to manage currencies and monitor exchange rates from the Frankfurter API.

---

# Features

## Currency Master Data

- View Currency List
- Add Currency
- Edit Currency
- Delete Currency
- Search Currency
- Filter Active / Inactive
- Validation
  - Required Fields
  - Currency Code must be exactly 3 characters
  - Currency Code must be unique

---

## FX Rate Monitor

- Refresh Exchange Rate
- Refresh All Exchange Rates
- Latest Exchange Rate
- Previous Business Day Rate
- Change Percentage
- Highlight if exchange rate changes more than ±1%
- Error Handling
